# Инструкция по настройке индексации в Google

## Что было сделано:

1. ✅ Создан `sitemap.xml` в папке `public/`
2. ✅ Обновлен `robots.txt` с указанием на sitemap
3. ✅ Улучшены SEO-метатеги в `index.html`

## Шаги для настройки в Google Search Console:

### 1. Подтверждение владения сайтом
- Перейдите в [Google Search Console](https://search.google.com/search-console)
- Добавьте ресурс (ваш сайт: `https://luchik.by`)
- Выберите метод подтверждения (рекомендуется HTML-тег или файл)

### 2. Отправка Sitemap
После подтверждения:
- Перейдите в раздел **"Sitemaps"** в левом меню
- Введите URL sitemap: `https://luchik.by/sitemap.xml`
- Нажмите **"Отправить"**

### 3. Запрос индексации
- Перейдите в раздел **"Проверка URL"**
- Введите URL главной страницы: `https://luchik.by/`
- Нажмите **"Запросить индексацию"**

### 4. Проверка robots.txt
- Перейдите в раздел **"robots.txt Tester"**
- Убедитесь, что файл доступен и разрешает индексацию

## Важно:

1. **После деплоя** убедитесь, что файлы доступны:
   - `https://luchik.by/sitemap.xml`
   - `https://luchik.by/robots.txt`

2. **Обновляйте sitemap.xml** при добавлении новых страниц

3. **Проверьте настройки сервера** - убедитесь, что:
   - Все маршруты ведут на `index.html` (для SPA)
   - Статические файлы доступны
   - Нет блокировок для поисковых роботов

## Проверка доступности:

После деплоя проверьте в браузере:
- `https://luchik.by/sitemap.xml` - должен открываться как XML
- `https://luchik.by/robots.txt` - должен показывать содержимое

## Время индексации:

- Обычно Google индексирует сайт в течение 1-7 дней
- После отправки sitemap проверка может занять несколько часов
- Используйте "Запрос индексации" для ускорения процесса
